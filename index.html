Introdução ao HTML na prática

(Voltar)

O HTML (HyperText Markup Language) é a linguagem de marcação responsável por estruturar e organizar os conteúdos de páginas na web. Ele não é uma linguagem de programação, mas sim uma linguagem de marcação que define os elementos e suas funções dentro de um site.

Conceitos importantes

Tags: São os elementos básicos do HTML. Elas definem o início e o fim de um conteúdo ou funcionalidade.
Atributos: São informações adicionais dentro das tags que configuram seu comportamento ou aparência.

Tags Básicas

  1. !DOCTYPE html – Declara que o documento é HTML5.
  2. html – Elemento raiz do documento.
  3. head – Contém metadados, título e links para arquivos externos.
  4. title – Define o título da página.
  5. body – Onde todo o conteúdo visível da página é inserido.
  6. h1 a h6 – Títulos, sendo h1 o mais importante.
  7. p – Parágrafo de texto.
  8. a – Cria um link.
  9. img – Insere uma imagem.
  10. ul, ol, li – Listas não ordenadas, ordenadas e itens.
  11. div e span – Elementos genéricos de agrupamento.

Estruturando seu HTML + Formatação

(Voltar)

A estruturação no HTML organiza os conteúdos da página de forma lógica e acessível.

  • Div – Agrupa e estrutura seções maiores.
  • Span – Agrupa pequenos trechos em linha.
  • Fieldset – Agrupa elementos em formulários.
  • Iframe – Incorpora outra página HTML.
  • Embed – Incorporação (em desuso).

Resenha sobre cores

No HTML, as cores podem ser aplicadas usando códigos hexadecimais (#FF5733), RGB, RGBA e HSL. Cada método permite personalizar a aparência dos elementos.

Trabalhando com Mídias

  • Img – Insere imagens com src, alt e title.
  • Audio – Insere áudio com controls, autoplay, loop.
  • Video – Insere vídeos com controls e track para legendas.

Entendendo HTML Semântico

(Voltar)

O HTML semântico utiliza tags que representam o significado do conteúdo, melhorando acessibilidade e SEO.

Vídeo explicativo

  • Accessibility Tree – Representação acessível do conteúdo.
  • alt – Descrição alternativa para imagens.
  • ARIA – Atributos de acessibilidade para elementos interativos.
